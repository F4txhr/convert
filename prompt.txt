VPN CONFIG CONVERTER PROJECT

DESKRIPSI:
Buatkan aplikasi VPN Config Converter yang lengkap menggunakan Python dengan fitur web interface dan Telegram bot integration. Aplikasi ini harus bisa mengkonversi berbagai format URL VPN ke format config yang berbeda-beda.

REQUIREMENTS:

1. STRUKTUR PROJECT:
   ```
   project/
   ├── app.py                 # Main Flask/FastAPI app
   ├── bot.py                 # Telegram bot
   ├── requirements.txt       # Python dependencies
   ├── config.py             # Configuration settings
   ├── templates/            # Config templates
   │   ├── singbox-basic.json
   │   ├── singbox-advanced.json
   │   ├── clash-minimal.yaml
   │   ├── clash-full.yaml
   │   ├── v2ray-custom.json
   │   └── xray-custom.json
   ├── parsers/              # VPN URL parsers
   │   ├── __init__.py
   │   ├── vmess.py
   │   ├── vless.py
   │   ├── trojan.py
   │   ├── shadowsocks.py
   │   └── hysteria.py
   ├── converters/           # Format converters
   │   ├── __init__.py
   │   ├── singbox.py
   │   ├── clash.py
   │   ├── v2ray.py
   │   └── base.py
   ├── utils/               # Utilities
   │   ├── __init__.py
   │   ├── base64_helper.py
   │   ├── template_engine.py
   │   └── validators.py
   ├── static/              # Web assets
   │   ├── css/
   │   ├── js/
   │   └── img/
   └── templates_web/       # HTML templates
       ├── index.html
       └── layout.html
   ```

2. FITUR UTAMA:

   A. WEB INTERFACE:
   - Upload/paste VPN URLs (vmess, vless, trojan, ss, ssr, hysteria, tuic)
   - Pilih format output (Singbox, Clash, V2ray, Xray, dll)
   - Pilih template yang sudah disediakan
   - Preview hasil konversi
   - Download sebagai file config
   - Generate QR code untuk subscription

   B. TELEGRAM BOT:
   - Command /convert untuk konversi URL
   - Command /template untuk list template available
   - Command /help untuk bantuan
   - Support file upload config
   - Inline query untuk quick convert
   - Admin panel untuk manage templates

   C. API ENDPOINTS:
   - POST /api/convert - Convert VPN URLs
   - GET /api/templates - List available templates
   - POST /api/validate - Validate config
   - GET /api/subscription - Convert subscription URL

3. PROTOKOL VPN YANG DIDUKUNG:
   INPUT:
   - vmess:// (VMess protocol)
   - vless:// (VLESS protocol)
   - trojan:// (Trojan protocol)
   - ss:// (Shadowsocks)
   - ssr:// (ShadowsocksR)
   - hysteria:// (Hysteria v1)
   - hysteria2:// (Hysteria v2)
   - tuic:// (TUIC protocol)
   - hy2:// (Hysteria2 alias)

   OUTPUT:
   - Singbox (JSON format)
   - Clash (YAML format)
   - Clash Meta (YAML format)
   - V2ray (JSON format)
   - Xray (JSON format)
   - Shadowrocket (text format)
   - QuantumultX (text format)
   - Surge (text format)
